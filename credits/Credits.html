Unity-Chan

Mondeto uses animation data of Unity-Chan. Because Mondeto uses part of data for characters other than Unity-Chan, the Unity Asset Store Terms of Use and EULA applies. However, the source code repository of Mondeto contains (redistributes) entire data of Unity-Chan, licensed under Unity-Chan License Terms. The license files (in English and Japanese, full and summarized versions) are inside docs/ThirdPartyLicenses/UCL2.0 directory.

There are some modified files (paths are relative to root directory of UnityChan asset).

  • Animators/UnityChanLocomotions_IK.controller: Based on Animators/UnityChanLocomotions.controller, but IK Pass is enabled.

Earth image (earth.jpg)

Earth texture image (Assets/StreamingAssets/scene_data/earth.jpg) is a resized version of “June, Blue Marble Next Generation w/ Topography” by NASA (National Aeronautics and Space Administration).

Skybox image (kloofendal_48d_partly_cloudy_2048x1024.jpg)

Skybox texture image (Assets/StreamingAssets/scene_data/kloofendal_48d_partly_cloudy_2048x1024.jpg) is Kloofendal 48d Partly Cloudy by Greg Zaal (licensed under CC0). This image is a resized version of the 8K tonemapped JPG.
